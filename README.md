Task 2 — Churn Analysis Conducted customer retention analysis on 7,043 Telco subscribers using Python (pandas) and Excel, identifying a 26.5% overall churn rate and $139K in monthly revenue at risk Discovered month-to-month contract customers churn at 42.7% vs 2.8% for two-year contracts — the single strongest predictor of churn in the dataset
